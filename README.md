# Todo App

> Generated by **ForgeAI** — AI-powered full-stack application generator.

## Features

- Secure sign‑up, login, and JWT token issuance
- Create, read, update, delete tasks with fields like title, description, due date, status, and assignee
- Group tasks into named lists; lists can belong to a user or a team
- Full‑text search on task titles/descriptions and filter by status, due date, assignee

## Quick Start

### Prerequisites

- Python 3.11+
- Node.js 18+ (for frontend)

### Backend

```bash
cd app
pip install -r requirements.txt
uvicorn main:app --reload
```

API available at: `http://localhost:8000`
Interactive docs: `http://localhost:8000/docs`

### Frontend

```bash
npm install
npm run dev
```

Frontend at: `http://localhost:5173`

### Docker

```bash
docker build -t todo_app .
docker run -p 8000:8000 todo_app
```
